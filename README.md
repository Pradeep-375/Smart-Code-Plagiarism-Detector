## Smart-Code-Plagiarism-Detector
📄🔍 Plagiarism Detection System built with Java to compare code source  files and detect similarities using Longest common subsequence(LCS) algorithm  

## 🚀 Features
1. 📄 Compare two or more documents
2. 🔍 Detect text similarity
3. ⚡ Fast string comparison algorithm
4. 💻 Simple and easy-to-use interface

## 🛠️ Technologies Used
1. Java

## 📂 Project Structure
Plagiarism-Detector/
│
├── src/
├── frontend/
├── README.md
└── main.java

## ▶️ How to Run
1. Clone the repository
2. Open the project in VS Code or IntelliJ
3. Compile the Java files
4. Run the main program

## Example:
javac PlagiarismDetector.java
java PlagiarismDetector

## 📸 Output
The program compares two documents and displays the similarity result.
